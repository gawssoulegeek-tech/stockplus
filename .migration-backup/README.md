
# 🚀 StockPlus - Votre Allié Commercial Intelligent

StockPlus est une plateforme SaaS premium conçue pour moderniser le commerce au Sénégal. Elle remplace les cahiers papier par une interface numérique intuitive propulsée par l'IA Awa.

## 🌟 Points Forts de cette version
- **Design Pop & Moderne** : Identité visuelle forte en Orange #FF8800.
- **Navigation Cinétique** : Menu immersif avec animations GSAP (responsive).
- **Tarification Locale** : Forfaits adaptés à 10 000 FCFA (Basic) et 25 000 FCFA (Pro).
- **IA Awa Intégrée** : Scan de produits, extraction de factures et analyses métier.
- **Inventaire Photo** : Possibilité d'ajouter des photos manuellement ou via scan.

## 📥 Comment Pousser vos modifications sur GitHub

Pour mettre à jour votre dépôt GitHub avec les derniers changements de design et de fonctionnalités, ouvrez votre terminal et exécutez ces commandes :

```bash
# 1. Préparer tous les fichiers modifiés
git add .

# 2. Enregistrer les changements avec un message descriptif
git commit -m "Mise à jour StockPlus : Design Pop, Navigation Kinetic et Tarification Pro"

# 3. Envoyer officiellement sur GitHub
git push
```

## ⚙️ Configuration des Variables d'Environnement

Assurez-vous que votre fichier `.env.local` contient les clés suivantes :

### AI & Genkit
* `GEMINI_API_KEY` : Votre clé API Google AI Studio.

## 🛠 Stack Technique
- **Frontend** : Next.js 15, React 19, Tailwind CSS.
- **Animations** : GSAP, Framer Motion, Lottie.
- **Backend** : Firebase SDK.
- **IA** : Google Genkit & Gemini 2.0.
